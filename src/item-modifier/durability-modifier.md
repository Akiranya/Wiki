# Durability Modifier

## Format
```yaml
durability: <durability>
damage: <durability>
```

## Description
This modifier will set the `<durability>` of the item

## Example
```yaml
durability-armor:
  NAME: '&aDamaged armor'
  LORE:
    - 'This armor is damaged.'
  ID: diamond helmet
  DAMAGE: 100
  POSITION-X: 1
  POSITION-Y: 1
```
