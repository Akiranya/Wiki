# Lore Modifier

## Format
```yaml
lore: <lore>
```
```yaml
lore:
- <lore>
- <lore>
- <lore>
...
```

## Description
This modifier will set the `<lore>` of the item

## Example
```yaml
stone-button:
  slot: 1
  id: stone
  name: "&cThis is a stone button"
  lore:
  - "This is just a stone button"
  - "&uNothing special"
```
