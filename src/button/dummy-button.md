# Dummy Button

## Format
```yaml
button-name:
  type: dummy # aliases: raw
  modifier1: <value1>
  modifier2: <value2>
  modifier3: <value3>
  ...
```

## Description
This is a decorative button. You can only set the display item of the button through [Item Modifiers](../Item-Modifier.md)

## Example
```yaml
dummy-button:
  type: dummy
  id: STONE
  name: "&eThis is a dummy button"
```
